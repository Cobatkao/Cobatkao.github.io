# 4.解构赋值

解构赋值语法是一个JavaScript表达式，它可以使用数组映射和对象文本构造的语法从数组和对象中提取值，对变量进行赋值。

var {foo, bar} = {foo: 'lorem', bar: 'ipsum'};
// foo => lorem and bar => ipsum
