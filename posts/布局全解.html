# 碎碎念

布局基本是初级前端面试必问的考点，快点来温习一遍吧！

本篇博文分常见的双栏，三栏布局来讲解，直接给出解决方案。只贴出了关键代码块，具体实现点击预览链接。

其实个人比较推荐双栏布局用float+bfc，三栏用float+margin，这也是最好理解的两种方法。

动手开始吧！

# 双栏自适应

即左侧固定，右侧自适应两栏布局

  • float+创建BFC

原理： 利用了左侧浮动，但是右侧盒子通过overflow: auto;形成了BFC，因此右侧盒子不会与浮动的元素重叠。

预览

  <style>
    .side {
      width: 220px;
      min-height: 600px;
      float: left;
      margin-right: 10px;
    }
    .main {
      min-height: 700px;
      overflow: hidden;
    }
  </style>

  • flex最简洁的方案

原理：两栏都设置为flex，左边栏flex: 0 0 auto;，意思是flex-grow: 0，默认不放大；flex-shrink: 0;，空间不足时也不缩小，固定大小；flex-basis: auto;，默认值为auto，即项目的本来大小；

也就是把侧栏固定大小。

右边栏flex: 1 1 auto;，占满剩余宽度，且可自适应大小。

预览

<style>
    .wrap {
          display: flex;
          align-items: flex-start;
        }
    .side {
      width: 220px;
      min-height: 600px;
      flex: 0 0 auto;
    }
    .main {
      min-height: 700px;
      flex: 1 1 auto;
    }
</style>

  • 双inline-block+calc

原理：通过calc()来动态计算主栏的宽度，所以需要知道侧栏的宽度及主栏距离左边的距离，技巧是对两栏用box-sizing: border-box;，方便计算。同时：作为两个inline-block的盒子，必须设置vertical-align来使其顶端对齐。另外，为了准确地应用计算出来的宽度，需要消除div之间的空格，需要通过设置父容器的font-size: 0;或者注释掉空格！

要点：

  • 消除inline-block的间隙bug：font-size: 0;
  • 对inline-block顶端对齐：vertical-align: top;
  • 需要知侧栏宽度与距离，用calc动态计算主栏宽度，实现自适应

预览

<style>
    .wrap {
      font-size: 0;
    }
    .side {
      width: 220px;
      min-height: 600px;
      display: inline-block;
      vertical-align: top;
      margin-right: 10px;
      box-sizing: border-box;
    }
    .main {
      min-height: 700px;
      display: inline-block;
      width: calc(100% - 230px);
      vertical-align: top;
      box-sizing: border-box;
    }
  </style>

  • 双float+calc

原理：与双inline-block类似，都是通过动态计算宽度来实现自适应，优势在于不用解决inline-block缝隙问题，不用设置顶端对齐。

缺点在于，应用了浮动，所以父元素要加clearfix，同时也需要盒子的宽度，设置box-sizing，方便动态计算主栏宽度。

预览

<style>
    .clearfix::after {
      content: '';
      display: block;
      clear: both;
    }
    .side {
      float: left;
      box-sizing: border-box;
      margin-right: 10px;
    }
    .main {
      float: left;
      box-sizing: border-box;
      width: calc(100% - 230px)
    }
  </style>

  • float+margin-left

原理：摒弃了上两种方法都要用calc来计算主栏宽度，利用了**block级别的元素盒子的宽度具有填满父容器，并随着父容器的宽度自适应的流动特性。**为了让右侧盒子和左侧盒子保持距离，需要为左侧盒子留出足够的距离。这个距离的大小为左侧盒子的宽度以及两个盒子之间的距离之和。然后将该值设置为右侧盒子的margin-left。

要点： - 侧栏清除浮动 - 需要计算右侧盒子的margin-left

预览

<style>
    .clearfix::after {
      content: '';
      display: block;
      clear: both;
    }
    .side {
      float: left;
    }
    .main {
      min-height: 700px;
      margin-left: 240px;
    }
</style>

  • absolute+margin-left

原理：另外一种让两个block排列到一起的方法是对左侧盒子使用position: absolute的绝对定位。这样，右侧盒子也能无视掉它

预览

<style>
    .side {
      position: absolute;
    }
    .main {
      min-height: 700px;
      margin-left: 240px;

# 三栏布局

所谓三栏布局就是指页面分为左中右三部分然后对中间一部分做自适应的一种布局方式。

  • absolute+margin

原理：最简单的一种三栏实现，两边栏绝对定位，中间用margin调整宽度自适应

预览

<style>
    html, body {
      padding: 0;
      margin: 0;
      height: 100%;
    }
    .left, .right {
      width: 150px;
      min-height: 450px;
      position: absolute;
      top: 0;
    }
    .left {
      left: 0;
      border: 3px solid green;
    }
    .right {
      right: 0;
      border: 3px solid red;
    }
    .main {
      min-height: 600px;
      margin: 0 160px 0 160px;
      border: 3px solid #000;
    }
  </style>

该方法有个明显的缺点，就是如果中间栏含有最小宽度限制，或是含有宽度的内部元素，当浏览器宽度小到一定程度，会发生层重叠的情况。

  • 圣杯布局

原理：body标签设置padding-left和right，为侧栏留出空位，左中右都float：left，两边栏用负边距挤上去，最后侧栏相对定位，挪到正确的空间内；

预览

  1. 中间部分需要根据浏览器宽度的变化而变化，所以要用100%，这里设左中右都向左浮动，因为中间100%，左层和右层根本没有位置上去；
  2. 把左层margin负100后，发现left上去了，因为负到出窗口没位置了，只能往上挪
  3. 按第二步这个方法，可以得出它只要挪动窗口宽度那么宽就能到最左边了，利用负边距，把左右栏定位
  4. 但由于左右栏遮挡住了中间部分，于是采用相对定位方法，各自相对于自己把自己挪出去，得到最终结果
  • 双飞翼布局

原理：圣杯布局实际看起来是复杂的后期维护性也不是很高，在淘宝UED的探讨下，出来了一种新的布局方式就是双飞翼布局，代码如上。增加多一个div就可以不用相对布局挪动侧边栏，中间栏在body上设置左右padding留空间了，只用到了浮动和负边距。和圣杯布局差异的地方已经被注释。

预览

双飞翼明显自适应效果更好，圣杯容易乱掉！

  • 浮动+margin

最简洁的方法，也足够理解，推荐！

预览

//注意元素次序
<div class="left">Left</div>
<div class="right">Right</div>
<div class="main">Main</div>

<style>
//CSS reset
    body,html {
        height:100%;
        padding: 0;
        margin: 0
    }
    //左栏左浮动
    .left {
        background: red;
        width: 100px;
        float: left;
        height: 100%;
    }
    //中间自适应
    .main {
        background: blue;
        height: 100%;
        margin:0px 200px 0px 100px;
    }
    //右栏右浮动
    .right {
        background: red;
        width: 200px;
        float: right;
        height: 100%;
    }
</style>
